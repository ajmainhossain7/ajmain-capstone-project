# Ajmain Capstone Project

A clean starting point for the capstone project repository, designed for future development and documentation.

## Tech Stack

- **Runtime:** Node.js
- **Package Manager:** npm
- **Version Control:** Git
- **Documentation:** Markdown

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm (comes with Node.js)

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd ajmain-capstone-project
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Available Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |
| `npm test` | Run tests (if configured) |
